BAO4 – Des textes aux graphes

La dernière partie de ce projet va nous permettre de générer des graphiques à partir des fichiers que nous avons construits. Plus particulièrement, nous allons utiliser le fichier corpus-titre-description.udpipe.xml que nous avons créé lors de la BàO précédente afin de visualiser ces données à partir d'un graphe.


Code Python

Ce script se lance de cette manière : python3 BAO4.py corpus-titre-description.udpipe.xml| curl -X POST -H 'Content-Type: text/csv' --data-binary @- "https://padagraph.magistry.fr/post_csv/graphe-culture".




On pourrait également potentiellement afficher les résultats du programme dans un fichier texte (objet.txt) : python3 BAO4.py corpus-titre-description.udpipe.xml > objet.txt. Ce format correspond en fait à un format .csv, le format nécéssaire pour afficher le graphe sur padagraph.
